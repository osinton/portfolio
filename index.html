About

The Sustainable Ship Configurator is a tool created with Business Finland for showcasing and comparing different sustainability solutions in the Finnish maritime industry. The website is interactive and different components of the ship can be changed while monitoring the emission changes.

Process

The task for this project was to create a tool to showcase the sustainability solutions but the method to display the data was for the team to decide. We came up with multiple different options and together with the client decided on the result.


This project was created with Next.js framework using TypeScript, JavaScript, CSS, Next.js, and React with a team of five people. As a leader for this project my key responsibilities were keeping the project on the road and in schedule. I also took part in designing the overall appearance as well as programming different components.

I also learned how to be a team leader and lead the team to the finish line with great results. The client was very pleased with the outcome of this project and we also got a great grade for this six month capstone university course.

AR Solar System

For viewing our solar system in your own home!

Technologies
C#
Unity
ARCore
About

This AR Solar System application is created for an AR/VR course for university with a project partner. In our solar system there are eight planets orbiting around the sun as well as around their own axis. We also added four of the largest moons in our solar system. Each of the planets, moons, and the Sun can be interacted with by pressing on them, which opens an image that tells the user some information about the celestial bodies. The image can be viewed even when putting the device down and the image will be closed only by pressing on the image. This is very convenient since this way them user doesn’t have to hold the device up the entire time.

Process

We created an AR Solar System on ARFoundation using C#, Unity, and ARCore. In addition to Unity we also utilized Canva and Blender with making graphics and assets for the project. In this project I was mostly responsible for coding and also putting together assets and scripts in Unity. My project partner did modeling for some of the assets and we both contributed in making the graphics.

Learnings

This project was my first time using the Unity engine and during this project I learned the basics of it quite well. One of the biggest difficulties for us was getting the UI elements to recognize touch, therefore we spent a lot of time on this aspect and also learned a lot about it. Searching for information on how to do certain parts of an application also took a great portion of our time with the project. We learned to utilize various different sources of information as well as use our own deduction skills to combine different forms of information.

Sustainable Ship Configurator

A tool for showcasing and comparing different sustainability solutions in the Finnish maritime industry.

Technologies
JavaScript
CSS
Next.js
React
Website
https://sus-ship.utu.fi/
About

The Sustainable Ship Configurator is a tool created with Business Finland for showcasing and comparing different sustainability solutions in the Finnish maritime industry. The website is interactive and different components of the ship can be changed while monitoring the emission changes.

Process

The task for this project was to create a tool to showcase the sustainability solutions but the method to display the data was for the team to decide. We came up with multiple different options and together with the client decided on the result.


This project was created with Next.js framework using TypeScript, JavaScript, CSS, Next.js, and React with a team of five people. As a leader for this project my key responsibilities were keeping the project on the road and in schedule. I also took part in designing the overall appearance as well as programming different components.

Learnings

This was my first project using JavaScript, Next.js, and React, and I learned a lot about them during the project. I also learned how to be a team leader and lead the team to the finish line with great results. The client was very pleased with the outcome of this project and we also got a great grade for this six month capstone university course.

The Stanley Parable Retelling

A (small) 2D version of The Stanley Parabel game

Technologies
C#
Unity
Website to download from
https://jaahas.itch.io/stanley-retelling
About

A 2D version and a retelling of The Stanley Parable game I made for my Master's Thesis. The purpose of this project was to reproduce important elements in the original Stanley Parable game in a different format and have people test the game and give feedback regarding their gaming experience. This is a much smaller version of the original game and the full playthrough of this version takes about 15 minutes.

Process

The Stanley Parable Retelling is built with Unity game engine using Visual Studio and C# programming language. The making of this game combined the use of different assets, Unity’s own building tools, programming, and combining different elements.

Learnings

Before this project, I had never used C# or Unity for more than a few practice projects. I learned how the mechanics of a 2D top-down game work as well as how to incorporate my ideas in the implementation of a game. This was also the largest project I had ever built and took a lot of time and patience. .
